# Subway Clone

Clone de interface inspirado no site/aplicativo da Subway, desenvolvido com foco em prática de front-end, reprodução visual de layout e aprimoramento das habilidades em construção de interfaces web.

## Sobre o projeto

O **Subway Clone** é um projeto criado com o objetivo de recriar a identidade visual e a experiência de interface inspiradas na marca Subway, servindo como exercício prático de desenvolvimento front-end.

A proposta do projeto é treinar a construção de layouts modernos, organização de seções, estilização visual e fidelidade a uma referência real de interface.

## Objetivos

- Praticar desenvolvimento front-end
- Reproduzir uma interface inspirada em uma aplicação real
- Melhorar habilidades em HTML, CSS e JavaScript
- Trabalhar organização visual e responsividade
- Desenvolver atenção a detalhes de layout e design

## Funcionalidades

- Interface inspirada na identidade visual da Subway
- Estruturação de seções e componentes visuais
- Elementos interativos com JavaScript
- Layout organizado e moderno
- Aplicação prática de conceitos de responsividade

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
